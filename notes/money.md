# Inkwell Money-Making Plan

## Main idea

Inkwell should not feel like a cash grab. The app is about personal stories, memory, taste, and identity, so the money model should feel clean and fair.

The best plan is:

1. Affiliate links first
2. Free account for normal users
3. Plus plan for serious users
4. Patron plan for supporters and profile customization
5. Ads only later, and only if needed

The goal is to let people use Inkwell for free, then make money from users who want deeper tools, better organization, profile customization, and ways to support the platform.

---

## 1. Affiliate links first

Affiliate links should be the first monetization method because they do not block users from using the app.

When someone searches or opens a story page, Inkwell can show helpful links like:

* Buy the book
* Buy the manga
* Buy the comic
* Watch the movie
* Find where to stream
* Support the creator

Example:

A user opens a book page.

Inkwell shows:

“Where to find it”

* Bookshop
* Amazon
* Local library link
* Publisher link

If the user buys through one of those links, Inkwell earns a small commission.

This works because the user already has intent. They are already looking at that story, so a “where to read/watch/buy” link is useful instead of annoying.

Affiliate links should not be the main focus of the page. They should be placed quietly on story detail pages, search result pages, and public profile shelves.

Best places for affiliate links:

* Story detail page
* Search results
* Public profile shelves
* “Planned” list
* Recommendation sections
* Creator/studio pages

Do not make the app feel like a store. Inkwell should stay a library/profile app first.

---

## 2. Free plan

The free plan should be strong enough that people actually want to join.

Free users should get:

* Search stories
* Add stories to library
* Track books, manga, comics, anime, movies, and shows
* Use basic statuses
* Rate stories
* Save a limited number of quotes, moments, characters, and notes
* Basic profile
* Public/private profile choice
* Follow people
* Explore public profiles

The free plan is important because Inkwell needs users first. Nobody will pay before they understand the value.

The free plan should answer:

“Can I use this app and understand why it matters?”

The answer should be yes.

---

## 3. Plus plan

The Plus plan is for serious users who want more control over their library and profile.

Recommended price:

$4.99/month
or
$39.99/year

Plus should include:

* Unlimited library entries
* Unlimited quotes
* Unlimited moments
* Unlimited character saves
* Unlimited notes
* Advanced stats
* Advanced filters
* Custom shelves
* Export library as CSV
* Private collections
* Custom profile themes
* No ads if ads are added later

Plus should feel useful, not cosmetic only.

The best Plus features are the ones that make the app better every day.

---

## 4. Advanced stats

Advanced stats show users patterns in their reading and watching life.

Examples:

* Stories completed this month
* Most saved format
* Highest-rated format
* Most paused format
* Drop rate
* Completion rate
* Average score by format
* Most saved creators/studios
* Most used tags
* Most quoted stories
* Stories with the most notes
* Favorite genres
* Monthly progress
* Yearly recap

Inkwell-specific stats could be more emotional:

* Stories that stayed with you most
* Most saved moments
* Most saved characters
* Most quoted stories
* Notes by mood
* Stories you keep returning to

This fits Inkwell because the app is not only about tracking. It is about meaning.

---

## 5. Advanced filters

Advanced filters help users find specific things inside a large library.

Free filters can be basic:

* Format
* Status

Plus filters can be deeper:

* Score
* Genre
* Creator/studio
* Date added
* Date completed
* Has quotes
* Has notes
* Has moments
* Has characters
* Public/private
* Custom shelf
* Language
* Reread/rewatch
* Tags

Example:

“Show me completed manga rated 9/10 or higher with saved quotes.”

Another example:

“Show me planned movies from Studio Ghibli that I have not rated yet.”

Advanced filters are valuable because basic filters are enough for a small library, but not enough for a serious user with hundreds of saved stories.

---

## 6. Custom shelves

Custom shelves let users organize stories in their own way.

Default statuses answer:

“Where am I with this story?”

Examples:

* In Progress
* Completed
* Planned
* Paused
* Dropped

Custom shelves answer:

“What does this story mean to me?”

Examples:

* Changed me
* Comfort stories
* Best endings
* Favorite villains
* Made me cry
* Need to reread
* Watch with friends
* Dark fantasy
* Childhood favorites
* Masterpieces

A story can only have one status, but it can belong to many shelves.

Example:

Vagabond
Status: Completed
Shelves: Changed me, Best art, Favorite protagonist

This is one of the strongest paid features because it connects directly to Inkwell’s emotional identity.

---

## 7. Export library as CSV

CSV export lets users download their data.

The export could include:

* Title
* Creator/studio
* Format
* Status
* Score
* Date added
* Date completed
* Custom shelves
* Tags
* Notes count
* Quotes count
* Moments count
* Characters count
* Public/private setting

This builds trust because users know their library is not trapped inside Inkwell forever.

Export is not the flashiest feature, but serious users will care about it.

---

## 8. Private collections

Private collections are hidden personal groups.

They are different from custom shelves.

Custom shelves can be part of someone’s public taste.

Private collections are for things the user wants to keep only for themselves.

Examples:

* Stories I do not want public
* Personal healing stories
* Gift ideas
* School project
* Private thoughts
* Stories that helped me
* Comfort collection

Free users should still have basic privacy.

Plus users get more detailed privacy control.

Best rule:

Basic private profile = free
Advanced private collections = Plus

This keeps the app trustworthy while still giving paid users extra control.

---

## 9. Custom profile themes

Custom profile themes let users make their profile feel personal.

Free profile:

* Basic layout
* Basic avatar
* Basic bio
* Default theme

Plus profile customization:

* Accent colors
* Profile banner
* Featured shelves
* Pinned stories
* Custom profile layout
* Special badges
* Theme presets
* Public/private sections

Theme examples:

* Dark Library
* Ink & Paper
* Cinema Glow
* Manga Panel
* Cozy Shelf
* Minimal Black
* Fantasy Archive

This matters because Inkwell is partly about identity. If users care about how their profile represents them, some will pay to make it feel more personal.

---

## 10. Patron plan

The Patron plan is not mainly about utility. It is about supporting Inkwell and getting extra identity/customization perks.

Recommended price:

$7.99/month
or
$69.99/year

Patron should include everything in Plus, plus:

* Patron badge
* Founding supporter badge
* Extra profile themes
* Early access to new features
* Vote on upcoming features
* Beta testing
* More profile customization
* Special profile effects
* Custom profile background
* Name highlight or profile frame

Patron should feel optional. Users should not feel forced into it.

Plus is for power features.
Patron is for support, identity, and early access.

This is important because some users will pay because they love the product and want it to survive.

---

## 11. Ads later

Ads should not be the first money method.

Ads can make money, but they usually need a lot of traffic before they matter. They can also make the app feel worse if they interrupt the emotional experience.

If Inkwell ever adds ads, they should be:

* Only for free users
* Not on private note pages
* Not inside writing/editing flows
* Not between emotional saved content
* Not invasive
* Not based on selling personal data

Better ad placements:

* Search results sidebar
* Public explore pages
* Story detail page sidebar
* Non-private recommendation areas

Plus and Patron users should get no ads.

Ads should be a backup revenue stream, not the main business.

---

## 12. What not to do

Inkwell should not:

* Sell personal user data
* Lock the whole product behind payment
* Show aggressive ads
* Put ads inside private notes
* Make users pay before they understand the value
* Make affiliate links feel like spam
* Make the free plan useless

The brand should feel trustworthy.

---

## Final money strategy

Start with:

Free account
Affiliate links
Optional Plus plan later

Then add:

Patron plan
More profile customization
Ethical ads only if needed

The best order is:

1. Build the free product
2. Add affiliate links naturally
3. Add Plus for serious users
4. Add Patron for supporters
5. Add ads only if the app needs more revenue

Inkwell should make money by helping users build a better library, not by making the free app annoying.
